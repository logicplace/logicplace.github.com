Command Line Options

These come directly after the dice command and before all expressions. Therefor like:

dice --ct 1 < 2?1:never

They affect output only. The rule to remember for command line options is that when you use lower-case it means "attempt to resolve value" where upper-case means "do not resolve value". A resolution will only fail if an item of higher precedence inside of it will not be resolved.
