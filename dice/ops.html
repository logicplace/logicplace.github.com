In the following, # will represent where numbers go and '' will represent where strings go. When a string is regarded as a number in the description, it will mean the length of that string (as if +'' had been run).

Unary

Ensure a number is positive: + #

Convert a string to an integer of its length: + ''

Toggle negativity of a number: - #

Convert a string to a negative integer of its length: - ''

Return 0 if the value is nonzero, 1 if zero with: !,~, or not

Ensure the type is string: @ # or @ ''

Ensure the type is number: # #

Convert a string to a number: # ''

Examples:

+1 = 1 +-1 = 1 +abc = 3 -1 = -1 --1 = 1 -abc = -3 ! = 1 !0 = 1 !3 = 0 !!3 = 1 @1 = '1' @'a' = 'a' #1 = 1 #'10' = 10 #'1,234' = 1234 #'4asdasd3' = 43

Binary

Addition

Add two numbers together: # + # or # + ''

Concatenate two strings: '' + ''

Concatenate a string and a number: '' + #

Examples:

1 + 1 = 2 1 + ab = 3 ab + c = abc ab + 1 = ab1

Subtraction

Subtract two numbers: # - # or # - ''

Cut off the right of a string: '' - #

Cut off the left of a string: '' - -#

Remove all occurances of a substring from a string: '' - ''

Examples:

2 - 1 = 1 2 - a = 1 abcd - 1 = abc abcd - -1 = bcd banana - a = bnn

Multiplication

Muliply two numbers: # * # or # * ''

Concatenate a string to itself a number of times (note: up to 20): '' * #

Count number of occurances of substring in a string: '' * ''

Examples:

3 * 4 = 12 3 * ab = 6 a * 3 = aaa banana * a = 3

Division

Divide two numbers: # / # or # / ''

Cut off the left of a string: '' / #

Cut off the right of a string: '' / -#

Return a string containing all occuraces of substring: '' / ''

5 / 2 = 2 abcd / 1 = bcd abcd / -1 = abc banana / a = aaa

Modulo

Divide two numbers and return the remainder: # % # or # % ''

Return a formatted string: '' % '' or '' % #

The above is slightly complex in function. In summary, it will replace both singular instances of % and intances of %- (with no % touching the left of it) with the substring, and then replace all sequences of % with a sequence of one less %

Examples:

5 % 2 = 1 a % 5 = 1 bcd % 4 = 3 a%c%%-%%%--%%%%% % b % d % e % f % g = abcde-g

Just as a quick explanation of the formatting example, here is what it looks like at each step:

a%c%%-%%%--%%%%% % b = abc%-%%--%%%% abc%-%%--%%%% % d = abcd%--%%% abcd%--%%% % e = abcde-%% abcde-%% % f = abcde-% abcde-% % g = abcde-g

Bitwise And

Bitwise operations involving integers are based on their binary forms.

For: # & # or # & ''

&01
000
101

Return a string with only the "on" chars: '' & #

Return a string with only the characters that are in both strings: '' & ''

Examples:

3 & 2 = 2 abc & 1 = a abc & 5 = ac abc & acd = ac

Bitwise Xor

For: # ^ # or # ^ ''

^01
001
110

Return a string with only the "off" chars: '' ^ #

Return the first string with all characters in the second string removed: '' ^ ''

Examples:

3 ^ 2 = 1 abc ^ 1 = bc abc ^ 5 = b abc ^ acd = b

Bitwise Or

For: # | # or # | '' or '' | #

|01
001
111

Return the two strings concatenated, with all characters in the first string removed from the second: '' | ''

Examples:

3 | 2 = 3 abc | acd = abcd

Comparison

Compare two values in some way, and usually return a boolean value (0 or 1)

Equivalency

These are the only comparators that test strings based on their actual content, instead of their length.

Test if two numbers are equivalent: # equ # or # equ '' or '' equ #

Test if two strings are equivalent: '' equ ''

equ can be: =,==,eq, or equ

Test if two numbers are not equivalent: # neq # or # neq '' or '' neq #

Test if two strings are not equivalent: '' neq ''

neq can be: !=,~=,ne, or neq

Examples:

1 == 2 = 0 2 == 2 = 1 abc == abc = 1 abc == cbd = 0 1 != a = 0

Magnitudal

These check the direction of inequivalence and etc. Strings are lost in this process.

Test if a number is greater than another number with: >,gt, or gtr

The above, or if they're equivalent: >=,ge, or geq

Test if a number is smaller than another number with: <,lt, or lss

The above, or if they're equivalent: <=,le, or leq

2 > 1 = 1 1 > abc = 0 2 < 1 = 0 1 < abc = 1

Selective

Compare each value and return one of them. These compare numbers only, but will return the actual string should it be the winning value.

Warning: Testing most of these comparisons as true or false values is not advised.

Return the smaller of the two or 0 if equivalent: <>,mn, or min

Return the smaller of the two: <=>,me, or meq

Return the larger of the two or 0 if equivalent: ><,mx, or max

Return the larger of the two: >=<,xe, or xeq

Return the left value if it is larger than the right, otherwise 0: >>

Return the left value if it is larger than or equal the right, otherwise 0: >=>

Return the left value if it is less than the right, otherwise 0: <<

Return the left value if it is less than or equal the right, otherwise 0: <=<

Examples:

1 <> 2 = 1 1 >< abc = abc 2 >> 1 = 2 2 << 1 = 0

Boolean

These are called boolean And, Or, and Xor (eXclusive OR), respectively.

If both values are nonzero, return the latter: && or and

If the first value is nonzero, return that, otherwise return the latter: || or or

Shorthand for !!# != !!#: ^^ or xor

Examples:

1 && 2 = 2 0 && 1 = 0 1 && 0 = 0 1 || ab = 1 0 || ab = ab 2 ^^ 0 = 1 2 ^^ 5 = 0 0 ^^ 3 = 1

Indexing

Indexing is only used for strings and certain variables. Attempting anything else will be ignored. Indexing always starts at 1.

Variable indexing is like arrays. More information in the variables section.

String indexing can be done in three ways:

Number index - Returns the character at this position in the string, starting from the left.

Negative number index - Returns the character at this position in the string, starting from the right.

String index - Returns the position of the first match in the string, otherwise 0.

Examples:

abc1 = a abc-1 = c abc8 = abcbdab = 2 abcbdae = 0
